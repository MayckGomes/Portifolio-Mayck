# 🏎️ Meu Portfólio

## Bem-vindo ao meu portfólio!

Este projeto representa um pouco da minha trajetória como desenvolvedor Mobile e Backend. Aqui reúno meus principais projetos, tecnologias, experiências e a forma como gosto de construir software.

A identidade visual foi inspirada na engenharia da Fórmula 1, utilizando conceitos como telemetria, grid, garagem e box para transmitir aquilo que mais valorizo no desenvolvimento de software: performance, precisão e evolução contínua.

## 🚀 Sobre mim

Sou desenvolvedor Mobile e Backend, apaixonado por transformar ideias em produtos completos.

Meu principal projeto é o DatePlan, um aplicativo para casais disponível na Google Play Store, desenvolvido de ponta a ponta por mim, desde a arquitetura do backend e infraestrutura até a experiência do usuário no aplicativo.

Tenho como foco o desenvolvimento utilizando Kotlin, Android e Kotlin Multiplatform, além de possuir experiência com backend em Java e Python, bancos de dados, infraestrutura e computação em nuvem.
